source.main.main()¶

Loads Json from path or string and calculates BMI and BMI categories. Optionally writes output.

Parameters

  • args ([str]) – BMI categories to be counted.

  • infile (str) – path to input. json file or json string.

  • outfile (str) – path to output file.

  • validate (bool) – flag for whether input data should be validated as possible human measurements.

Returns

A dict containing provided BMI categories as keys and their respective counts.
